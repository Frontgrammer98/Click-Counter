## Installation


Ensure you have Node.js and npm installed on your machine. Then, clone the repository and run:

```bash
npm install
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/Frontgrammer98/Click-Counter.git
```

Go to the project directory

```bash
  cd TypeScript
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev

  o + Enter
```
